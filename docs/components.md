# Компоненты и стандартные экраны

## Компоненты из коробки
- **phoneInput** — ввод номера телефона с автодетектом оператора и валидацией.
- **moneyInput** — компонент приёма денег, умеет считать купюры и делать возврат.
- **scenario** — связывает экраны в сценарий и ожидает событий `screen.next()` / `screen.back()`.

## Пример документации компонента
**phoneInput**
- Свойства: `operator`, `phone`
- События: `change`, `valid`
- Методы: `get()`, `set()`

**moneyInput**
- Свойства: `amount`, `billCount`, `canReturn`
- События: `income`, `canReturn`
- Методы: `cashBack()`

## Составные экраны
- **EnterPhoneNumber** — экран ввода номера с клавиатурой и кнопкой «Далее».
- **PayMoney** — экран приёма денег с отображением остатка и возврата.
