# scenario

## Назначение
Компонент оркестрации экранов. Служит входной точкой для сценария и управляет переходами.

## Контракт API
### Props
- `initialScreen`: string — стартовый экран.
- `state`: object — базовое состояние сценария.

### Events
- `screen.next()` — переход на следующий экран.
- `screen.back()` — возврат к предыдущему экрану.

### Methods
- `start()` — запуск сценария.
- `stop()` — остановка сценария.

## Пример использования
```json
{
  "type": "scenario",
  "props": { "initialScreen": "EnterPhoneNumber", "state": {} }
}
```
