# moneyInput

## Назначение
Компонент приёма денег, подсчитывает купюры, отображает остаток и управляет возвратом.

## Контракт API
### Props
- `amount`: number — сумма к оплате.
- `billCount`: number — количество принятых купюр.
- `canReturn`: boolean — доступность возврата.

### Events
- `income(amount)` — событие приёма средств.
- `canReturn()` — сигнал доступности возврата.

### Methods
- `cashBack()` — инициировать возврат.

## Пример использования
```json
{
  "type": "moneyInput",
  "props": { "amount": 0, "billCount": 0, "canReturn": false },
  "events": { "income": "screen.next" }
}
```

## Чек-лист качества
- [ ] Корректный подсчёт купюр и суммы.
- [ ] Обновление остатка в реальном времени.
- [ ] Наличие сценария возврата средств.
- [ ] Доступность: корректные роли и статусные сообщения.
- [ ] Тесты: обновление суммы, событие `income`.
