# phoneInput

## Назначение
Компонент ввода номера телефона с автодетектом оператора и встроенной валидацией.

## Контракт API
### Props
- `operator`: "auto" | string — режим определения оператора.
- `phone`: string — текущее значение номера.

### Events
- `change(phone)` — вызывается при изменении значения.
- `valid(phone)` — вызывается после успешной валидации.

### Methods
- `get()` — вернуть текущее значение.
- `set(phone)` — установить значение.

## Пример использования
```json
{
  "type": "phoneInput",
  "props": { "operator": "auto", "phone": "" },
  "events": { "valid": "screen.next" }
}
```
