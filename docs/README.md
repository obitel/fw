# FW Visual Flow Studio — Документация

## О продукте
FW Visual Flow Studio — визуальная среда, в которой интерфейсы собираются мышью, а сценарии превращаются в
экранные потоки, события и автоматические тесты. Пользователь создаёт приложение один раз и сразу получает
версии для web, desktop и терминалов.

## Ключевые принципы
- **Реактивная модель данных**: компоненты подписаны на единые сущности, изменения отражаются везде.
- **Каталог компонентов**: поиск, рейтинги, версии и стабильность на всех платформах.
- **Сценарии как макросы**: запишите цепочку действий и получите тест, который проходит в каждой сборке.
- **Документация из коробки**: каждый компонент имеет свойства, события, методы и карту данных.
- **Визуальный редактор**: канвас, инспектор, слои данных и событий в одной рабочей области.

## Локальная разработка
Этот репозиторий содержит лендинг и UI-слой на **React + Vite + Tailwind + shadcn/ui**, подключаемый к Laravel.

### Установка фронтенда
```bash
npm install
npm run dev
```

### Интеграция с Laravel
1. Убедитесь, что в Laravel-проекте установлен `laravel-vite-plugin`.
2. Подключите view `resources/views/app.blade.php` в `routes/web.php`.
3. Включите сборку через `@vite(["resources/css/app.css", "resources/js/app.tsx"])`.

## Планы реализации (MVP → v1)
1. **MVP**: визуальный редактор, базовые компоненты (ввод, кнопки, контейнеры), сценарный рантайм.
2. **v0.9**: библиотека шаблонов, редактор данных, генерация тестов.
3. **v1.0**: публикация в web/desktop/terminal, каталог компонентов, marketplace-метаданные.
